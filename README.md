# Darin — Childcare Management App

AI 돌봄 커뮤니케이션 앱 **React Native (Expo)** 프로토타입입니다.

## Running the app

```bash
pnpm install
pnpm start
```

- **iOS 시뮬레이터:** 터미널에서 `i` 키
- **Android 에뮬레이터:** `a` 키
- **실기기:** Expo Go 앱으로 QR 코드 스캔

```bash
pnpm run ios      # iOS 직접 실행
pnpm run android  # Android 직접 실행
pnpm run typecheck
```

자세한 구조와 기능은 [work.md](./work.md)를 참고하세요.
